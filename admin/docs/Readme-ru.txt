Утилита администрирования "Nailer"

Администратор "Nailer" это: 
добавление, изменение, удаление текстовой информации; 
полнотекстовый поиск; 
постраничный вывод информации; 
настройки выводимой по результатам поиска информации; 
управление дополнительными графическими элементами веб-сервера; 
"умное" удаление данных (возможность восстановления данных из корзины); 
отслеживание целостности данных в БД; 
архивирование всех данных; 
система разграничения прав доступа к данным для проектов с несколькими ответственными редакторами; 
журнал учета операций по каждому из ответственных редакторов; 
экспорт данных в Microsoft Excel; 
изменения цветовой гаммы; 
подробная статистическая информация о работе веб-сервера; 
управление форумом, гостевой книгой, опросником, чатом. 
Администратор "Nailer" дает возможность тонкой настройки и регулирования работы всего веб-сервера.
